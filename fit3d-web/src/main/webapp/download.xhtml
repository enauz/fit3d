Fit3D is available as API or standalone version. This allows to perform sophisticated calculations or to integrate Fit3D into custom processing workflows. Furthermore, there are some features available which are not implemented in the web version.


The Fit3D algorithm is part of the SiNGA framework which is developed in our lab. Please see the project homepage on how to use the API.
A standalone and platform-independent implementation of the Fit3D algorithm is available as runnable jar file. Please see the project homepage on how to use the API.
